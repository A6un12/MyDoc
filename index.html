Compress files

Shrink PDFs and images down toward a target file size you choose. PDFs are compressed by re-rendering each page as a compressed image — this is very effective on scanned or image-heavy PDFs, but it removes selectable text, since the page becomes a picture. For text-heavy PDFs where you need to keep selectable text, there's a floor to how small the file can get.

Click to choose files, or drop them here

Mix PDFs and images — each is compressed toward the same target, individually

Each file is compressed toward this size on its own. The tool searches for the highest quality that still fits — if the target is smaller than what's realistically achievable (e.g. a 50-page PDF into 100 KB), you'll get the smallest version it could produce instead, clearly labeled.
